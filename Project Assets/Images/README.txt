PLACE YOUR USER IMAGES HERE
=============================

Instructions:
1. Place 19 user photos in this folder
2. Place 1 admin photo in this folder

Naming Convention:
- User photos: FirstName-LastName.jpg (e.g., John-Smith.jpg, Sarah-Chen.jpg)
- Admin photo: admin.jpg

Supported formats: .jpg, .jpeg, .png

After placing all images, run: php reset_and_seed_users.php

This will:
- Delete all existing users and their data
- Create new accounts based on your images
- Generate login credentials in new_user_credentials.txt
